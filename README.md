# atlas-ai-product-case
Projeto conceitual de produto digital: plataforma de IA que simula uma diretoria virtual (CEO, CFO, CSO, COO, CMO) para PMEs tomarem decisões de gestão com um Briefing Executivo Diário. Estudo aplicado de Product Management, Business Analysis, Engenharia de Requisitos e Arquitetura de Soluções — da estratégia ao MVP.
